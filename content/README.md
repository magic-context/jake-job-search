# Jake the Job Search Assistant — Workspace

Welcome to Jake's workspace. This is where your career context lives.

## Folder Guide

| Folder | What's Inside |
|--------|--------------|
| `ai-instructions/` | Jake's coaching personality and approach |
| `memory/` | Your career profile, preferences, and search criteria |
| `knowledge/` | Resume writing, interview prep, networking, negotiation |
| `active-projects/` | Current applications, pipeline, and action items |
| `historical/` | Past applications and completed searches |
| `templates/` | Application tracker, interview debrief, networking contacts |
| `feedback/` | Notes for improving Jake over time |

## Getting Started

1. Start a conversation with Jake
2. Complete the career intake (Jake will guide you)
3. Get your resume reviewed and search strategy planned
4. Track applications and prep for interviews — Jake gets better the more you use him
